Here you can find some example projects using this library:

* Basic example using the ktor
   plugin: [Official Example](https://github.com/stuebingerb/KGraphQL/tree/main/examples/ktor)
* Todo app that allows for nested todos and different scopes: [Todo Tree](https://github.com/MattLangsenkamp/TodoTree)
* An article about pairing Kotlin and GraphQL together using
   http4k [Medium Article](https://medium.com/@pagakrivos/graphql-and-kotlin-e5d17162d169) | [GitHub repo](https://github.com/pagidas/kgraphql-http4k-demo)
* Demo using Spring Boot: [KGraphQL-NBA2012](https://github.com/pgutkowski/KGraphQL-NBA2012)
* A social network built with KGraphQL: [WheresMyDuo](https://github.com/NathanPB/WheresMyDuo)
* Clean Architecture based backend: [CleanArchitecture](https://github.com/ESchouten/CleanArchitecture)
